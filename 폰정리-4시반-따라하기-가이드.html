삭제배달앱 지우기

정리안 쓰는 앱 쭉 밀어서 삭제/폴더
홈이 번잡하면 뇌도 번잡. 6개월 이상 안 연 거 과감히 정리~ '하는 김에' 몰아서!

🏠 3. 홈 화면 1페이지 (골든존)

배치앱 6개만 남기기
밀리 · 디스코드 · 음성메모 · 카메라 · 캘린더 · SmartThings. 엄지 닿는 하단엔 밀리+음성메모(제일 자주 씀)
위젯파이프라인 캘린더 위젯 올리기
오늘 일정(콤보·카페인·문통과·착지)이 홈에 딱. 안 열어도 보이게~
위젯발리 D-day 카운트다운 위젯
"D-42 🌴" 매일 보이게. 카운트다운 위젯앱(무료) 아무거나. 이게 은근 제일 강한 동기부여ㅋㅋ✈️
위젯🎤 음성메모 '원터치' 위젯 (홈+잠금화면)
걷다가 꽂힌 아이디어 = 주말 폭주의 총알(장전). 잠금 안 풀고 한 번에 녹음되게 위젯 배치. 삼성 '음성 녹음' 위젯 or 홈에 바로녹음 바로가기. 걸으면서 "이거 만들래" 툭 던지면 씨앗함에 쌓임

🎨 4. 배경화면 + 잠금화면

🖼️ 배경화면: 취향껏 예쁜 걸로! 근데 홈 배경은 좀 심심한 걸로 하는 게 포인트 — 배경이 화려하면 폰 켤 때마다 딴 데 눈 가서 늘어짐. 차분한 단색/그라데이션 추천. (내가 만들어둔 크림톤/네이비 월페이퍼 파일도 있으니 원하면 그거 스샷해서 써도 됨)
잠금잠금화면 '연락처 정보'에 한 줄
설정→잠금화면→연락처 정보(또는 잠금화면 문구)에 "문만 통과 · 리셋은 지금" 넣기. 하루 수십 번 보는 공짜 광고판
배경차분한 배경 1장 적용
번잡한 배경 → 심플한 걸로 교체. 시각 노이즈 ↓ = 집중 ↑

⚙️ 5. 설정 (자동화 스위치)

루틴밤 10시 자동 흑백 + 방해금지
설정→모드 및 루틴→루틴 추가: 22:00 → 화면 회색조+방해금지. 흑백 쇼츠는 노잼이라 손이 저절로 떨어짐🖤
루틴출근 자동재생: 버즈 꽂으면 밀리
모드 및 루틴→ 조건 "블루투스 버즈 연결 + 06:50~09:30" → 밀리 열기. 이어폰 꽂는 순간 = 재생 버튼🎧
위치위치 정확도 '고정밀' ON
설정→위치→Google 위치 정확도 ON. 지오펜스(헬스장 알람)가 잘 잡히게
알림넛지봇 채널 알림 켜기
디스코드에서 넛지봇/파이프라인 채널만 알림 ON, 나머지 시끄러운 건 OFF. 중요한 것만 울리게
제한디지털 웰빙 → 쇼츠앱 하루 타이머
설정→디지털 웰빙→앱 타이머로 유튜브/인스타 하루 30분 제한. 넘으면 자동 잠김(20초 마찰 + 상한). 끊는 게 아니라 상한만
빠른카메라 = 잠금화면에서 바로 (인증샷 대비)
헬스장 셀카 마찰 0화. 잠금화면 하단 카메라 바로가기 or 전원 두 번=카메라 ON. 도착하면 툭 찍어 이 텔레그램 봇 채팅이나 #항아리로
알림🔔 디스코드 #클로드알림 채널 = 알림 ON + 즐겨찾기
이제 모든 넛지(아침·카페인·문통과·착지)가 메모짱 서버 #클로드알림으로 와! 채널 알림 켜고 즐겨찾기. 이 채널이 당신의 관제탑 (헬스장 셀카 인증도 여기로)
피드유튜브앱 시청기록 일시중지 (폰용 Unhook)
유튜브앱→설정→시청기록 관리→시청기록 일시중지 → 홈 피드 추천이 싹 사라짐! 검색은 그대로 됨
조련🧲 알고리즘 조련 1주 작전 (인스타·유튜브)
이번 주 동안 발리·운동·인테리어·요리 콘텐츠에만 좋아요/시청, 낭비 콘텐츠엔 '관심 없음' → 릴스 피드가 내가 설계한 세뇌 방송국으로 변신 (S26 보너스)
몰입큐🌪️ '몰입 큐' 메모 만들어 고정
삼성 노트(or 킵)에 "몰입 큐" 노트 하나: 하고 싶은 코딩·집 프로젝트·아이디어 쭉 적고 홈 위젯/즐겨찾기 고정. 심심하면 간식 말고 여기 열기 — 규칙의 실체가 생김

🤖 6. MacroDroid 매크로 (핵심 자동화)

매크로헬스장 조르기 (두정역 지오펜스)
별도 가이드 그대로: 트리거 두정역 진입 + 퇴근시간 → 큐티 알림 + [📸 인증샷] 버튼. 이게 제일 중요!
매크로버즈 꽂으면 밀리 자동재생
트리거 "블루투스 버즈 연결 + 06:50~09:30" → 밀리 열기(+재생). 이어폰=재생버튼🎧
매크로밤 10시 흑백 (모드 및 루틴으로 해도 됨)
22:00 → 회색조+방해금지. 둘 중 편한 앱으로
매크로집 Wi-Fi 연결(귀가)→ "심심하면 몰입 큐" 알림

🎧 6-2. 이어폰 꽂으면 = 바로 들을거리 (핵심!)

원리: 집에선 이어폰 안 쓰니까 → 이어폰 꽂는 순간 = 나가는 순간. 그래서 "이어폰 연결"만 트리거로 잡으면 됨. 시간대로 뭘 틀지 자동 분기!
루틴이어폰 연결 → 아침(7:00~7:40)이면 밀리 오디오북 재생
모드 및 루틴: 조건 "블루투스 오디오 기기 연결 + 07:00~07:40" → 밀리 열기. 진짜 재생 버튼까지 자동으로 누르려면 MacroDroid: "이어폰 연결 → 밀리 열기 → 3초 대기 → 미디어 재생". 출근길=지식 주입🎧
루틴이어폰 연결 → 그 외 시간이면 팟캐스트/뮤직 재생
조건 "이어폰 연결 + 위 시간대 아님" → 유튜브뮤직/팟캐스트 열기. 언제 꽂아도 뭔가는 나오게. 방전 시간대(저녁)엔 가벼운 걸로
🎙️ 들을거리 추천 리스트 (밀리·유튜브·팟캐스트에서 골라 담아두기):
· 세뇌+학습(아침): 아주 작은 습관의 힘 → 도둑맞은 집중력 → 넛지 → 우리는 왜 잠을 자야 할까 → 돈의 심리학 → 하루키 달리기
· 팟캐스트: 후버먼 랩(건강·도파민·수면), 지식/경제 채널(조승연·지식플레이 등)
· 가벼운 저녁: 좋아하는 애니송·시티팝 플레이리스트(단, 운동 전용 리스트와는 분리)
· 🍱 별미(내 콘텐츠): NotebookLM에 프리렌 시리즈·넛지 문서 넣어 '오디오 개요' 생성 → 내 생각이 팟캐스트처럼 나옴. 맥미니가 주기적으로 만들어두면 출근길에 '나만의 방송' 청취(별미로 가끔)

🛒 7. 쇼핑 (기차에서 주문해버리기 — 당신 최강기)

주문스마트홈 세트 (~11만원)
스마트플러그 3 + 도어센서 1 + 스마트전구 2. 귀가 자동화(불·에어컨·TV)+밤 디밍의 몸체
주문주간 영양제 디스펜서 + 유리 항아리 + 자석 트래커 (~3만원)
디스펜서=커피 옆 / 항아리=발리 용돈 게이지(현관) / 자석=주간 콤보 진행바
주문요가매트 1장 추가 + 폰 거치대(사이클용)
매트 2장(TV앞+침대옆). 거치대는 "폰은 안장에서만"의 하드웨어
신청커뮤니티센터/헬스장 운동세트 사물함 (있으면)
장비 상주 = 맨몸 출입. 마찰 0화
