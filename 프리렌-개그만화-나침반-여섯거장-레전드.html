여섯 거장 레전드썰 ㅋㅋ

"천재라서 성공한 거 아님?" → 알고 보니 다 개판이었던 사람들 手記
페른이 프리렌을 붙잡고 물었다…
0발단
페른: "성공한 사람들은 다 천재잖아요! 전 글렀어요!!" 프리렌: "(흔들흔들) 야… 걔네 대부분 인생 개판이었어… 놔봐…"
두구두구두구…
1🍔 레이 크록 — 52세 믹서기 아저씨
52세까지 믹서기 팔던 아저씨. 근데 주문서에 "햄버거집 하나가 믹서기 8대 주문" → "미친 거 아냐?" 하고 비행기 타고 직접 감 → 그게 맥도날드였음. 페른: "궁금해서 가본 게 제국의 시작…?!"
2🍗 커널 샌더스 — 62세 파산 할배
62세에 파산. 남은 건 월 105달러 연금 + 치킨 레시피 하나. 보통 여기서 인생 접는데 — 이 할배 차에 압력솥 싣고 전국 식당 순회하며 "레시피 사세요" 영업. 차에서 잠까지 잠. 페른: "62세 스타트업…레전드다 진짜 ㅋㅋㅋ"
잠깐, 더 웃긴 게 남음
3⚾ 무라카미 하루키 — 야구 보다 각성
29세 재즈바 사장. 어느 날 야구장에서 맥주 마시다가 타자가 2루타 치는 순간 — "오 나 소설 쓸 수 있을 것 같은데?" 뜬금 각성. 그날 밤 만년필 사서 씀. 페른: "계획성 0인데 노벨상 후보?! 사기캐잖아요!!"
4👰 베라 왕 — 40세, 빡쳐서 창업
올림픽 탈락 + 보그 편집장 탈락(17년 다님). 39세 결혼 준비하는데 맘에 드는 웨딩드레스가 세상에 없음 → 빡침 → "내가 만들고 만다" → 40세에 창업 → 지금 웨딩드레스=베라 왕. 페른: "빡쳐서 만든 게 세계 1위…?!"
5💿 데릭 시버스 — 실수로 대박 난 사람
기타리스트가 자기 CD 팔려고 결제 페이지 하나 만듦. 친구들이 "나도 껴줘" → 얼떨결에 회사됨 → 2200만 달러에 매각 → 전액 기부. 명언: "'와 대박!' 아니면 그냥 노(No)." 페른: "의도 안 했는데 성공한 케이스 실화냐구요"
6🧠 바버라 오클리 — 수학 0점의 반란
초·중·고 수학 줄줄이 낙제. "난 수학 뇌 아냐" 확신. 근데 26세에 "어? 뇌 바꿔볼까?" → 공대 교수됨 → "수포자가 어떻게 바뀌나"를 강의로 → 400만 명 수강 (지구 최대급). 페른: "약점을 팔아서 세계 1위… 반칙 아녜요?!"
그래서 결론은…
💥 6명 공통점 (충격)
천재 0명. 대신 전원: ①남의 거 부끄럼 없이 베낌 ②짜증·빡침·약점에서 시작 ③계획 없이 "어 이거다" 하고 저질러버림 ④그 뒤에 미친 듯이 반복. …어? 이거 완전 너(페른)잖아?
7🎬 결말
페른: "그럼… 저 빡쳐서 저질러버리는 거 잘하는데… 저도 되는 거예요?!" 프리렌: "이제 알았네. …근데 멱살은 놔줄래?"
천재는 없었다.
다들 빡치고, 베끼고, 저지르고,
존나 반복했을 뿐.

그거… 너도 할 수 있잖아? 😏
