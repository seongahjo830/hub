헬스장 도착하면
폰이 조르게 만들기 📸

MacroDroid로 "위치 감지 → 큐티 알람 → 셀카 인증" — 서버 없이 폰만으로
🎯 완성되면 이렇게 굴러가요 (당신 동선 반영)
퇴근 시간대에 두정역에 도착하면(= 헬스장 갈지 결심하는 바로 그 순간) → 폰이 삐릭 울리며 매번 다른 큐티 멘트로 "커뮤니티센터 고고!" → 알림의 [📸 인증샷] 버튼 → 커뮤니티센터 헬스장에서 셀카 → 디스코드 #항아리 공유 = 오늘 성공✅. 안 올리면 조용히 실패 기록 → 내일 두 배로 귀엽게 조름 ㅋㅋ🥺
왜 두정역이 트리거? 헬스장(커뮤니티센터)에 도착한 뒤엔 이미 늦어요 — 결심은 두정역 내려서 집으로 갈지 센터로 꺾을지 갈리는 그 순간에 나니까, 알람도 딱 거기서 울려야 발이 센터로 꺾여요. 인증샷만 커뮤니티센터에서.

🅰️ 준비 (2분, 지금)

STEP 1
MacroDroid 설치 (플레이스토어, 무료 — 매크로 5개까지 무료라 충분). 그리고 위치 권한 = "항상 허용" + 정확도(고정밀) 켜기. 설정 → 위치 → 위치 서비스 → 'Google 위치 정확도' ON.
STEP 2
배터리 최적화 예외 처리(중요!). 설정 → 앱 → MacroDroid → 배터리 → "제한 없음". 이거 안 하면 폰이 절전한다고 위치 감지를 꺼버려서 알람이 안 와요. (갤폴드는 특히 빡세게 절전해서 필수)
왜 이렇게? 지오펜스(위치 울타리)는 폰이 GPS·와이파이·기지국을 계속 살짝 봐야 작동해요. 권한 "항상" + 배터리 예외 = 정확한 위치 추적의 핵심이에요. 반경을 150m로 잡는 이유도 정확도 여유(너무 좁으면 놓침).

🅱️ 매크로 만들기 (5분)

STEP 3
MacroDroid → 매크로 추가(+) → 이름 "헬스장 인증 조르기".
STEP 4 · 트리거(언제)
트리거 추가 → 위치 → 지오펜스 트리거 → 지도 검색창에 "두정역" 입력(천안) → 핀 확인하고 반경 200m → "지역 진입 시". (대략 좌표 36.830, 127.139 — 근데 이름 검색이 제일 정확)
+ 트리거 하나 더 → 일/시간 → 요일 월·목·금 & 시간 18:00~20:30 (수요일 19:00~21:00, 토요일 12:30~14:00). = "두정역 도착 + 퇴근 시간"일 때만 울림.
STEP 5 · 동작(뭐 할지)
동작 추가 → 알림 → 제목 "📸 인증샷 찍자!" → 내용에 아래 멘트 리스트를 매크로드로이드 변수 리스트로 넣고 "랜덤 선택" (or 간단히: 동작 여러 개 만들어 랜덤 분기). → 알림에 버튼(액션): "찍기" 추가 → 그 버튼 동작 = 앱 열기: 카메라 (또는 디스코드 공유).
STEP 6 · 제약(선택)
제약 추가 → "하루 1회만"(같은 날 중복 알림 방지). 끝! 저장.

🅲 위치 추적 정확도 튜닝

잘 잡히게 하려면: ①반경 150m 유지(좁으면 놓침) ②실내(헬스장 건물)에선 GPS 약해서, 진입은 건물 도착 전 길에서 잡히게 반경 넉넉히 ③MacroDroid 설정 → 지오펜스 → "업데이트 주기"를 짧게 안 하면 배터리 아끼지만 반응 느림 → 보통 기본값이면 충분.
안 울리면 체크: ①배터리 최적화 예외 됐나(STEP 2) ②위치 "항상 허용"인가 ③고정밀 위치 ON인가 — 이 3개가 99%의 원인이에요.
📍 위치 확정 (반영 완료)
트리거 = 두정역(천안, 지도에서 "두정역" 검색) · 인증샷 = 커뮤니티센터(두정역 효정 헤링턴플레이스, 봉정로 324 — 당신 아파트 커뮤니티센터 헬스장). 반경 200m·시간대는 당신 근무(월목금 18-20:30 / 수 19-21 / 토 12:30-14)에 맞춰 세팅. 지도 검색이 좌표보다 정확하니 이름으로 찍으면 돼요.
